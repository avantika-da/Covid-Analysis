# Covid-Analysis
Using data to analysis Covid world wide effects. Tools used: Sql and Tableau.

This project delivers a comprehensive exploratory data analysis (EDA) and time-series assessment of the global COVID-19 pandemic. The analysis focuses on tracking key epidemiological metrics, identifying major trend shifts, and understanding the geographical spread and impact of the virus over time.

The goal is to provide a clear, data-driven narrative of the crisis, utilizing robust statistical methods and compelling data visualizations.
